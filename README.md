# 🎭 情绪解谜馆

> 治愈系轻量解谜微信小游戏

## 📖 项目介绍

**情绪解谜馆** 是一款治愈系轻量解谜小游戏，玩家通过解开简单谜题收集"情绪瓶"，解锁暖心故事，还可以将情绪瓶送给好友。

## 🎮 游戏玩法

1. **解谜** - 8个精心设计的单人谜题
2. **收集** - 解开谜题获得情绪瓶
3. **故事** - 收集情绪瓶解锁暖心故事

> ⚡ 纯单机体验，无需联网，无需微信

## 🛠️ 技术栈

- **前端**: 原生 JavaScript + HTML5 + CSS3
- **平台**: 微信小游戏
- **适配**: 自动适配各种屏幕尺寸

## 📁 项目结构

```
game-emotion-puzzle/
├── index.html    # 游戏入口页面
├── game.js       # 游戏核心逻辑

```

## 🚀 快速开始

### 方式一：浏览器预览

```bash
# 直接用浏览器打开 index.html 即可预览
# 或使用本地服务器
python3 -m http.server 8080
# 访问 http://localhost:8080
```

### 方式二：微信开发者工具

1. 下载 [微信开发者工具](https://developers.weixin.qq.com/minigame/dev/devtools/download.html)
2. 创建新项目，选择"小游戏"
3. 将本项目文件夹导入
4. 点击预览即可在微信中运行
